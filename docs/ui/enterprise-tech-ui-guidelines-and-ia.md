# 企业级科技风设计指导与关键页面 IA（UI Designer）

## 视觉风格
- 企业级、可信、专业、科技感
- 冷色系、清晰层级、数据密度高但可读

## 色彩与排版
- 主色：#2F6BFF
- 背景：#0F1420 / #F7F9FC
- 字体：PingFang SC / Inter

## 组件与布局
- 12栅格、卡片式布局
- 主按钮 36/40px，高对比

## 关键页面 IA
### A. 专家输入页
- 输入区、参数区、附件、合规提示、预览与校验

### B. 运营控制台
- KPI 总览、任务监控、运行警报、资源负载

### C. 产品包详情页
- 版本、依赖、权限、安全扫描、历史版本

### D. 运行时执行页
- 状态、进度、日志、性能指标、操作区
