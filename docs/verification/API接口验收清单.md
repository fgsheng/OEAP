# 接口层验收清单（API Tester）

> 覆盖维度：专家输入、产品包、运行时、审计、事件流

## 1) 范围与前置条件
- 环境：预发/测试环境，配置/权限/脱敏与生产一致
- 鉴权：管理员/普通/受限账号
- 依赖：外部服务可 Mock 或沙箱
- SLA：P95 < 200ms；错误率 < 0.1%

## 3) 验收标准
- 功能通过率 100%
- 安全：无高危漏洞
- 性能：P95<200ms，错误率<0.1%
- 事件：无丢失/乱序
- 审计：完整且权限生效
