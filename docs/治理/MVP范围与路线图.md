# MVP 范围与路线图

## 1. MVP定位
**目的**：验证“需求挖掘标准化 + 方案承接半自动化”是否显著提升效率与复用。
**原则**：可用、可验证、可扩展（不追求完美交付）。

## 2. MVP范围（0-3个月）
**核心功能**
1. 行业需求访谈模板库（电力/新能源优先）
2. 需求结构化录入 + 标签化
3. 方案承接框架生成（模板级）
4. 方案组件沉淀与复用推荐（基础规则）
5. 关键指标看板

**验收标准**
- 至少 5 个客户项目试用
- 需求整理效率提升 ≥30%
- 方案框架复用率 ≥30%

## 3. Next（3-6个月）
- 行业知识图谱扩展
- AI 辅助需求洞察（冲突/缺口提示）
- 方案模板参数化
- 方案复用率分析

## 4. Later（6-12个月）
- 智能方案生成（多轮优化）
- 深度行业场景扩展
- 与 CRM/项目管理系统集成
- 多企业知识库协作

## 5. 不做清单
- 全自动交付
- 全行业覆盖
- 复杂定制化需求
