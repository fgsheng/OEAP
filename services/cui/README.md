# CUI (Textual) Prototype

A Textual-based CUI prototype for the Enterprise AI Platform, covering:
- 专家沉淀
- 真实用户运行
- 销售工程师脚本

## Run

```bash
cd services/cui
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

## Output

Structured JSON files are saved in `services/cui/data/` after completing or saving a flow.
